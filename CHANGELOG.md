## [0.1.0] - 2025-01-01

### Added
- Initial MVP release
- Bunny companion webview panel
- `Bunny: Start Companion` command
- `Bunny: Run Current File` command for Python and JavaScript
- `Bunny: Reset Stats` command
- `Bunny: Toggle Sound` command
- Success animations (carrot eating + floating hearts)
- Error animations (sad/crying bunny)
- XP and level system
- Streak tracking with milestone bonuses
- Local persistence via VS Code globalState
- Configurable Python and Node.js commands
- Configurable pet name and sound settings
- Content Security Policy for webview
- Unit tests for core logic
- Integration tests for extension activation
- GitHub Actions CI pipeline
- VSIX packaging for Antigravity compatibility
